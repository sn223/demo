CSS

a. Vad är en regel i CSS och vad består den av?

b. Vad är skillnaden mellan id och class? När bör man använda id respektive class?

Svar till fråga a. / b.

a. Regler är det vi använder oss av inne i stil mallarna(CSS) för att t.ex bestämma hur en HTML tagg eller class ska se ut. Regler är uppbygda av olika selektorer. Detta kan vara element(t.ex div), attribut, id eller class. Det kan också i vissa fall vara en kombination av olika.

b. Skillnaden mellan att ge ett element en class och ett id är att idnamnet du ger ett elemnt får bara nämnas 1 gång medans en class kan du nämna samma namn hur många gånger du vill. Id använder du om du vill sätta ett namn på något specifikt element. Det går ju att anävnda id fler gånger men då måste det vara ett annat namn. Jag har valt att använda class i min kod eftersom det gör det lätare för mig att lägga samma regler på elementen som jag har använt då jag kan sätta samma namn på dem.
